

```bash
cd local

# PostgreSQL, Redis, LocalStack 시작
docker-compose up -d

# 로그 확인
docker-compose logs -f

# 상태 확인
docker-compose ps
```

**실행되는 서비스:**
- PostgreSQL: `localhost:5432`
- Redis: `localhost:6379`
- LocalStack (AWS 서비스 에뮬레이터): `localhost:4566`
- Prometheus: `localhost:9090`
- Grafana: `localhost:3000`

**2. Controller 작성 예시**
```java
@RestController
@RequestMapping("/api/v1/users")
@RequiredArgsConstructor
public class UserController {
    
    private final UserService userService;
    
    @GetMapping("/{userId}")
    public ResponseEntity<UserResponse> getUser(
        @PathVariable Long userId,
        @AuthenticationPrincipal UserPrincipal principal
    ) {
        // 비즈니스 로직은 Service에 위임
        User user = userService.findById(userId);
        return ResponseEntity.ok(UserResponse.from(user));
    }
}
```

**3. Service Layer**
```java
@Service
@Transactional(readOnly = true)
@RequiredArgsConstructor
public class UserService {
    
    private final UserRepository userRepository;
    
    public User findById(Long userId) {
        return userRepository.findById(userId)
            .orElseThrow(() -> new UserNotFoundException(userId));
    }
    
    @Transactional
    public User createUser(CreateUserRequest request) {
        // validation
        validateEmail(request.getEmail());
        
        // business logic
        User user = User.builder()
            .email(request.getEmail())
            .name(request.getName())
            .build();
            
        return userRepository.save(user);
    }
}
```

#### 주요 개발 규칙

**1. API Router 작성**
```python
from fastapi import APIRouter, Depends
from app.models.workout import WorkoutRecommendationRequest, WorkoutRecommendationResponse
from app.services.bedrock_service import BedrockService

router = APIRouter(prefix="/api/v1/workout", tags=["workout"])

@router.post("/recommend", response_model=WorkoutRecommendationResponse)
async def recommend_workout(
    request: WorkoutRecommendationRequest,
    bedrock_service: BedrockService = Depends()
):
    """
    AI 기반 운동 추천
    """
    result = await bedrock_service.generate_workout_recommendation(
        user_profile=request.user_profile,
        workout_history=request.workout_history
    )
    
    return WorkoutRecommendationResponse(
        recommendation=result,
        confidence_score=0.95
    )
```

**2. Pydantic 모델**
```python
from pydantic import BaseModel, Field, validator

class WorkoutRecommendationRequest(BaseModel):
    user_id: int = Field(..., gt=0)
    user_profile: dict
    workout_history: list[dict] = []
    
    @validator('user_profile')
    def validate_profile(cls, v):
        required_fields = ['age', 'weight', 'height', 'fitness_level']
        if not all(field in v for field in required_fields):
            raise ValueError(f"Missing required fields: {required_fields}")
        return v
```

**3. 설정 관리 (Pydantic Settings)**
```python
from pydantic_settings import BaseSettings

class Settings(BaseSettings):
    # Database
    postgres_host: str
    postgres_port: int = 5432
    postgres_db: str
    postgres_user: str
    postgres_password: str
    
    # AWS Bedrock
    bedrock_model_id: str = "anthropic.claude-3-sonnet-20240229-v1:0"
    bedrock_region: str = "us-east-1"
    
    # Redis
    redis_host: str = "localhost"
    redis_port: int = 6379
    
    class Config:
        env_file = ".env"

settings = Settings()
```

## 코드 작성 규칙

### 공통 규칙

1. **명명 규칙**
   - 클래스: PascalCase
   - 함수/변수: camelCase (Java), snake_case (Python)
   - 상수: UPPER_SNAKE_CASE
   - 파일: kebab-case (JS/TS), snake_case (Python)

2. **주석**
   - 복잡한 로직에는 why 설명
   - API는 docstring/JavaDoc 필수

3. **에러 처리**
   - 구체적인 예외 타입 사용
   - 에러 메시지는 사용자 친화적으로

### Java (Spring Boot)

```java
// 좋은 예시
@Service
@Transactional(readOnly = true)
@RequiredArgsConstructor
public class WorkoutService {
    
    private final WorkoutRepository workoutRepository;
    
    /**
     * 사용자의 운동 계획 조회
     * 
     * @param userId 사용자 ID
     * @return 운동 계획 목록
     * @throws UserNotFoundException 사용자를 찾을 수 없을 때
     */
    public List<Workout> findUserWorkouts(Long userId) {
        return workoutRepository.findByUserId(userId);
    }
}
```

### Python (FastAPI)

```python
# 좋은 예시
from typing import Optional
from fastapi import HTTPException

async def get_user_profile(user_id: int) -> dict:
    """
    사용자 프로필 조회
    
    Args:
        user_id: 사용자 ID
        
    Returns:
        사용자 프로필 딕셔너리
        
    Raises:
        HTTPException: 사용자를 찾을 수 없을 때 (404)
    """
    profile = await db.fetch_user(user_id)
    if not profile:
        raise HTTPException(
            status_code=404,
            detail=f"User {user_id} not found"
        )
    return profile
```
